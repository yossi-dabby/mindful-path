# Adolescents CBT Specialized Series - English

Approved English LTR specialized CBT worksheet library for adolescents / teens ages 12-18.

## Contents for app integration

- 60 individual English PDF worksheets
- 10 module-level PDFs, one full PDF per module
- manifest.adolescents-cbt-specialized-en.json
- README_EN.md

## Target path

public/forms/en/adolescents/cbt-specialized/

## Language behavior

These forms are English-only and must appear only when the active app language is English, unless the user explicitly requests English forms.

Do not show these forms in Hebrew or any other language view. Do not use them as fallback for missing translations.

## AI resolver requirement

The in-app AI must use the manifest metadata as the source of truth and match worksheets by therapeutic content, module, user intent, clinical keywords, audience, age range, and language - not only by title or filename.
