# פוביות ספציפיות - CBT לילדים בעברית

חבילה מקוצרת ל-Git, נוצרה כדי למנוע שגיאת Windows: Path too long.

## מה יש בתיקייה
- `1.2.1.pdf` עד `1.2.10.pdf` - עשרה טפסים בודדים.
- `all.pdf` - קומבינציה אחת של כל עשרת הטפסים.
- `manifest.json` - מטאדאטה בעברית עם כותרות ותיאורים.
- `target_path.txt` - נתיב יעד מומלץ בריפו.

## נתיב יעד מומלץ בריפו
`public/forms/he/children/cbt-specialized/specific-phobias/`

## חשוב
הקבצים בתוך ה-ZIP קוצרו בכוונה לשמות כמו `1.2.1.pdf` כדי לאפשר חילוץ רגיל ב-Windows בלי שגיאת path too long.
